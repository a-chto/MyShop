для удобства работаем в gitbash
посмотреть файлы и директории в текущей директории
ls 
ls -a // для отображения скрытых файлов

для создания директории (папки)
mkdir dir_name

имя файла или папки лучше писать в нижнем регистре
на английском, разделяя слова _

для создания файла
touch path_to_file.ext
touch dir_name/file.txt

для удаления файла
rm path_to_file/file_name.ext

для удаления папки
rm -rf dir_name

переход между директориями
cd path_to_dir
переход на уровень выше
cd ..

чтобы не вводить полностью название папки или файла
достаточно ввести несколько символов, то используем tab
например, я нахожусь в директории, где есть папка shop
cd s <нажимаю tab> и система подберет максимально подходящий
вариант
./shop/
./shot/
./sh/
cd s <жмем tab>
cd sh <жмем tab>
cd shop
