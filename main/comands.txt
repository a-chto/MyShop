для удобства работаем в gitbash
посмотреть файлы и директории в текущей директории
ls
ls -a // для отображения скрытых файлов

для создания директории (папки)
mkdir dir_name

имя файла или папки нужно писать в нижнем регистре on english, разделяя слова _

для создания файла
touch path_to_file.ext
touch dir_name/file.txt

для удаления файла
rm path_to_file/file_name.ext

для удаления папки
rm -rf dir_name

переход между директориями
cd path_to_dir
переход на уровень выше
cd ..

чтобы не вводить полностью название папки или файла
достаточно ввести несколько символов, о используем tab
например, я нахожусь в директории, где есть папка shop
cd s <нажимаю tab> и система подберет максимально подходящий вариант
./shop/
./shot/
./sh/
cd s <жмем tab>
cd sh <жмем tab>
cd shop

django команды
любую команду начинаем символов
python manage.py [command_name]
создание приложения: startapp [app_name]
создание миграций: makemigrations
применение миграций: migrate
запуск локального сервера: runserver
запуск сервера в локальной сети: runserver 0.0.0.0

если ошибка [Errno 2] No such file or directory
то проверяем в какой диреткории мы находимся
